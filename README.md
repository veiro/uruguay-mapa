https://uruguay-mapa-como-usar.herokuapp.com/

## Como ejecutar?

1- Clonar
```
git clone https://github.com/veiro/uruguay-mapa.git
```


2- Instalar paquetes
```
npm install
```

3- Iniciar
```
npm start
```

Acceder a la url http://localhost:4200/
